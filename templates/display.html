Key Notes For CNN


1. Original Canvas size is 800px*800px*3channels(R,G,B)

2. Canvas is converted to jpeg image which is 28*28*3channels(R,G,B)

3. Then jpeg image underoges through 2 filter (using openCV)

4. Then image is transformed to [1,28,28,1] 4d numpy array

5. Then it undergoes several layers (Convolutionlayer, padding layer, maxpooling layer, ANN layer with ReLU activation function) which is pretrained using tensorflow.

6. Accuracy is better than ANN.

Key Notes For ANN


1. Original Canvas size is 800px*800px*3channels(R,G,B)

2. Canvas is converted to jpeg image which is 28*28*3channels(R,G,B)

3. Then jpeg image underoges through 2 filter (using openCV)

4. Then image is transformed to [1,28*28] 2d numpy array

5. Finally it is given as input to ANN which is pre trained with thousands of handwritten data.

6. As training data is not big so model accuracy is not good.
